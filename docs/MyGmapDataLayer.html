MyGmapDataLayer

Each MyGmapDataLayer is made up of a Google Maps Data Layer object as well as the MyGmapDataLayer.on, MyGmapDataLayer.off, MyGmapDataLayer.onHover, MyGmapDataLayer.onClick, and MyGmapDataLayer.style methods. To learn more about Google Map Data Layers, see https://developers.google.com/maps/documentation/javascript/datalayer.

(static) .onClick(limitopt, callbackopt)

This method creates a Google Maps listener that sets the 'selected' property of a MyGmapDataLayer feature to 'true' when a user clicks it. When a geometry feature is clicked, it remains in a selected state until it is clicked again. Calling this method with an integer parameter sets a limit for the total number of simultaneously selected geometry features. To learn more about Google Map listeners, see https://developers.google.com/maps/documentation/javascript/events.
Parameters:
Name Type Attributes Description
limit Number <optional>
 This optional parameter can be used either to set a limit for the total number of simultaneously selected features, by entering an integer value, or to turn off the listener, by entering the string 'OFF'.
callback function <optional>
 This is an optional callback to send 'event.feature' to.
