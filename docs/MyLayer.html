MyLayer

Each MyLayer consists of the two store members MyLayer.data and MyLayer.geom, and the configuration members MyLayer.breaks, MyLayer.classes, MyLayer.classification, MyLayer.colourscale, MyLayer.domain, MyLayer.geostats, and MyLayer.status.

(static) .views

This method gets or sets a new view based on MyLayer's MyLayer.data and MyLayer.geom stores. Views associated with MyLayer share the same MyLayer.data and MyLayer.geom but can each visualize the values in those stores in unique and complimentary ways.

(static) .breaks()

This method sets the number of breaks for the data in MyLayer. This setting, along with MyLayer.classification and MyLayer.colourscale constitute the core GIS visualization settings. See http://www.ncgia.ucsb.edu/cctp/units/unit47/html/comp_class.html for general information about geospatial classification and groupings.

(static) .colourscale()

This method is used to set a colour scale and to calculate colours for individual data points based on that scale. socioscapes includes support for all valid colourbrew colour scales http://colorbrewer2.org/. This setting, along with MyLayer.breaks and MyLayer.classifications constitute the core GIS visualization settings. See http://www.ncgia.ucsb.edu/cctp/units/unit47/47_f.html for general information about geospatial visualization.

// returns the hexadecimal value for '100' given the ColourBrew spectrum 'YlOrRd' and five breaks.

(static) .data()

This method sets the data store of the the associated MyLayer. If the fetch is succesful, MyLayer.status('data') and MyLayer.status('geostats') will both return true. If no parameters are provided, the method returns the currently stored data values, if any exist.The method can take two parameters, the first can be a string name for any valid socioscapes data fetcher, a function that returns a valid socioscapes-data-object, or a valid socioscapes-data-object. The second parameter should be an object that provides all necessary configuration options for the first.

(static) .geom()

This method sets the geom store of MyLayer. If the fetch is successful, MyLayer.status('geom') will return true.
