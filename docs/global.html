fetchGoogleAuth(config, callback) → {Object}

This function requests authorization to use a Google API, and if received, loads that API client. See http://developers.google.com/api-client-library/javascript/reference/referencedocs.
Parameters:
Name Type Description
config Object An object with configuration options for Google APIs.
Properties
Name Type Description
auth Object Configuration options for the auth request (eg. .client_id, .scope, .immediate)
client.name Object The name of the Google API client to load.
client.version Object The version of the Google API client to load.
callback function This is an optional callback that returns the result of the client load.
